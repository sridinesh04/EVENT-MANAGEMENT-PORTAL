# Event Management Portal

A role-based web application designed to manage events efficiently, allowing administrators to create and manage events while enabling students/users to register and participate.

## 🚀 Features
- Role-based access (Admin & User)
- Event creation, update, and deletion
- User registration for events
- Dashboard for managing event data
- Secure data handling using backend validation

## 🛠️ Tech Stack
- Backend: Node.js, Express
- Database: MySQL
- Frontend: HTML, CSS, JavaScript (if used)

## 📊 Key Highlights
- Handles 100+ user records efficiently
- Optimized database queries for faster data retrieval
- RESTful APIs for seamless communication between frontend and backend
